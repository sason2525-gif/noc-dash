# ControlShift - דשבורד NOC מסונכרן

מערכת ניהול משמרות NOC מתקדמת עם סנכרון נתונים מלא בין כל הבקרים.

## איך מתחילים? (הגדרת Firebase)

כדי שהנתונים יישמרו ויסתנכרנו בין מחשבים שונים, יש לחבר את המערכת ל-Firebase בחינם:

1. **יצירת פרויקט:** היכנסו ל-[Firebase Console](https://console.firebase.google.com/) וצרו פרויקט חדש.
2. **הפעלת Database:** 
   - בתפריט הצדדי בחרו ב-**Firestore Database**.
   - לחצו על **Create Database**.
   - בחרו במיקום הקרוב אליכם (למשל `europe-west3`).
   - בחרו ב-**Start in Test Mode** כדי לאפשר כתיבת נתונים ללא הרשאות מורכבות.
3. **קבלת מפתחות:** 
   - בהגדרות הפרויקט (Project Settings), הוסיפו אפליקציית Web.
   - העתיקו את אובייקט ה-`firebaseConfig`.
4. **חיבור הקוד:** הדביקו את הנתונים בקובץ `firebaseService.ts` בתוך משתנה ה-`firebaseConfig`.

## תכונות המערכת
- **סנכרון Real-time:** ברגע שבקר אחד מעדכן סטטוס, הוא משתנה אצל כולם מיד.
- **סיכום AI:** כפתור סיכום המשתמש ב-Gemini כדי ליצור דו"ח טכני נקי (ללא אמוג'ים).
- **מותאם לעברית:** ממשק RTL מלא.
- **הכנה ל-WhatsApp:** כפתור ייצוא מהיר להודעת ווטסאפ מעוצבת.

## העלאה ל-GitHub
לאחר שסיימתם להגדיר את המפתחות, תוכלו להעלות את הפרויקט ל-GitHub Pages או לכל שרת אחר והוא יעבוד כאתר חי לכל דבר.
